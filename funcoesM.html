Funções quadraticas

Conceito:

Função modular é a função que representa os elementos em módulos. Em que o módulo pode ser representado pelas barras nos valores positivos. Os numeros dentro dos módulos sempre serão positvos, por conta desse fator, é necessario considerador que o valor fora do módulo pode assumir negativo ou positivo.

Exemplo:

|x| = 5

  • --> x = +5

  • --> x = -5


    • Exemplo:

    Na equação modular: |x + 3| = 5, pode-se assumir os valores:

  • |x + 3| = 5 --> x = 2

  • |x + 3| = -5 --> x = -8

Equação Modular

A equação modular é aquela em que a variavel está dentro do módulo. E para sua resolução, resolve-se como qualquer outra função.

Exemplos:


a) |2x-1| = 3

Com valor positivo:

2x - 1 = 3

2x = 3 + 1

2x = 4

x = 2


Com valor negativo:

2x - 1 = -3

2x = - 3 + 1

2x = -2

x = -1


b) |x²- 9| = 5

Com valor positivo:

x² - 9 = 5

x² = 5 + 9

x² = 14

x = +- raiz(14)


Com valor negativo:

x² - 9 = -5

x² = -5 + 9

x² = 4

x = +- 2

Função modular

É chamada de função modular a função de ℝ em ℝ, tal que f(x) = |x|.

Em que |x| pode assumir valores de:

  • |x| = x sendo maior ou igual a 0

  • |x| = -x sendo menor que 0

Gráfico

Para fazer um gráfico manualmente é necessario seguir as instruções abaixo:

1° - Fazer um gráfico com base a equação dentro do módulo


Vou usar como exemplo, para uma melhor explicação a função:

f(x)= -|x-2| + 3

2° - Passar, em forma de reflexão, a parte que está negativa na função para a positiva



3° - Obesrvar se há um sinal de soma ou subtração fora do módulo, e assim mover/refletir o módula na direção do sinal



4° - Observar se há algum elemento somando ou subtranindo o módulo, e assim aumentar os subtrair a quantidade indica na imagem.
